# HTML Visualizer & AppWrap Pro v6

A powerful standalone web application for visualizing code in 50+ programming languages with device mockup frames, live preview, and AI agent integration.

## Features
- 🎨 **50+ Languages** with syntax highlighting and renderable previews
- 📱 **AppWrap Pro** device mockups (iPhone, Android, iPad, Laptop, Desktop)
- 🔄 **Multi-tab Editor** (HTML/CSS/JS)
- 📊 **Streamlit Simulator** for Python Streamlit code
- 🤖 **Send to Agent** via AutoGPT External API
- 📦 **PWA Generator** for offline-capable apps
- ⚡ **Emmet Support** for rapid HTML expansion
- 🔄 **Export** to React/Vue/Svelte components
- 📱 **QR Code** preview sharing
- 🚀 **GitHub Pages Deploy** built-in
- 🤗 **Hugging Face Spaces Deploy** built-in
